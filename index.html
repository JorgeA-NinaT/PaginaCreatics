💡 Encender un LED

🎯 Objetivo: Aprender a encender un LED en el pin digital 13 del Arduino Uno y comprender los conceptos básicos de salidas digitales.
🛠️ Lista de materiales necesarios
Arduino Uno R3
LED Rojo 5mm
Resistencia 220Ω
Protoboard 400 puntos
Cables Jumper M-M
📐 Esquemático del circuito

Conexión básica: Arduino Pin 2 → Resistencia 220Ω → LED → GND

💻 Código fuente completo 
// =====================================
// El pin 2 es la salida digital que usaremos para encender el LED
#define LedPin 2

void setup() {
  // pinMode configura el pin digital como OUTPUT (salida).
  pinMode(LedPin, OUTPUT);

  // digitalWrite escribe el estado de una salida digital (LOW o HIGH)
  digitalWrite(LedPin, LOW);  // LOW = estado bajo (0V), el LED se apaga
}

void loop() {
  // Encendemos el LED:
  digitalWrite(LedPin, HIGH);  // HIGH = estado alto (5V), el LED se enciende
}
// =====================================
